MP3元数据修改工具
简介

一款简单易用的MP3文件标签编辑和编码修复工具，解决中文MP3乱码问题。

主要功能

🔧 修复MP3标签乱码

⚡ 批量处理多个文件

✏️ 手动编辑元数据

▶️ 内置播放器

📊 编码质量分析

快速开始

下载并运行 MP3TagEdit.exe

点击"加载文件"选择MP3文件

右键文件选择"修复乱码"

或使用"批量修复"处理多个文件

系统要求

Windows 7/10/11

.NET 9.0 Desktop Runtime

50MB 可用空间

使用说明
基本操作

加载文件: 点击工具栏第一个按钮

播放测试: 双击文件或点击播放按钮

修复乱码: 右键文件 → 修复乱码

手动编辑: 右键文件 → 手动编辑

批量操作: 多选文件后右键操作

颜色说明

🟢 绿色: 元数据正常

🔴 红色: 需要修复

⚫ 黑色: 未分析
